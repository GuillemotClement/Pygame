# Pygame

Projet realiser pendant le parcour BootDev.

## Lancer le projet

- Cloner le projet 

- Creer un environnement virtuel 
```shell
 python3 -m venv venv
```

- Installer les dependances
```shell
pip install -r requirements.txt
```

- Lancer le projet
```shell
python3 main.py
 ```
